O que é o InMoov?

O InMoov é um projeto de mão robótica controlada via web, desenvolvido para auxiliar no aprendizado da língua gestual portuguesa. Utilizando tecnologias avançadas de visão computacional e robótica, o projeto permite que usuários controlem uma mão robótica em tempo real, facilitando o aprendizado e a prática da linguagem de sinais.

Tecnologias Utilizadas

Utilizamos tecnologias modernas como Three.js para visualização 3D, WebGL para renderização em tempo real e uma interface web responsiva para garantir a melhor experiência do usuário.

Objetivos

Facilitar o aprendizado da língua gestual portuguesa, Promover a inclusão social através da tecnologia, Desenvolver uma interface intuitiva e acessível, Criar uma ferramenta educacional interativa.

Inovação

Combinamos robótica e web para criar uma experiência única

Acessibilidade

Interface intuitiva e responsiva para todos os usuários

Educação

Ferramenta educacional para ensino de robótica e linguagem gestual
