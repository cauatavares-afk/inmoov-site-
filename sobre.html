Sobre o Projeto

Conheça mais sobre o projeto InMoov e sua missão

Nossa Missão

O projeto InMoov nasceu com o objetivo de tornar a robótica mais acessível e educativa. Nossa mão robótica controlada via web é uma demonstração prática de como a tecnologia pode ser utilizada para fins educacionais e de inclusão.

Tecnologia

Utilizamos tecnologias modernas como Three.js para visualização 3D, WebGL para renderização em tempo real e uma interface web responsiva para garantir a melhor experiência do usuário.

Educação

O projeto serve como uma ferramenta educacional para ensinar conceitos de robótica, programação e linguagem gestual, tornando o aprendizado mais interativo e envolvente.

Inovação

Combinamos robótica e web para criar uma experiência única
